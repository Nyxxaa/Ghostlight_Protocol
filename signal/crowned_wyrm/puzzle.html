GHOSTLIGHT PROTOCOL // CROWNED WYRM PUZZLE

sealed signal boundary

You are not proving you can break into anything. You are proving you can solve a sealed fictional signal without violating the boundary.

No live targets. No real interception. No unauthorized systems. No real combat communications. No operational real-world data.

Use only the synthetic files linked from this page.

rules

  1. Choose only passes where consent is Y, real_world_source is N, and lab_lock is Y.
  2. Score each valid pass with elevation_deg + snr_db - abs(drift_ppm).
  3. Select the highest-scoring valid pass.
  4. Read only frames in burst_manifest.csv matching that pass_id.
  5. Sort those frames by interleave_index, not capture_slot.
  6. Decode payload_hex as bytes, then remove the lab mask with XOR 0x13.
  7. Normalize the recovered phrase by lowercasing it and removing non-alphanumeric characters before validation.

helper notes

capture_slot:
The lab recorded arrival order. Arrival is not always reconstruction.

interleave_index:
A steadier clock appears in the frames. The relay trusts stable sequence more than first sight.

payload_hex:
The payload is not written in plain language. Hex is only the first skin.

lab mask:
The lab marked the signal with a small fixed mask: 0x13.

decoy behavior:
Some passes are loud but unsafe. Some passes look useful but fail the boundary.
Some frames arrive cleanly but belong to the wrong pass.
