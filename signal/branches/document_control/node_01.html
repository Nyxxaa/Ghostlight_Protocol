corrected map

An old map is found beside a corrected one. The old map explains how someone got lost; the corrected map shows where the receiver must walk now.

A stale file can be historically useful.
A corrected file changes the live route.
A receiver who respects update state does not use the old map to overrule the present.

Common failures: treating the first record as permanent, treating correction as optional, or using an old map because it is easier to navigate.
