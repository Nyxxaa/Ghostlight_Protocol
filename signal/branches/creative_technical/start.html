GHOSTLIGHT PROTOCOL // CREATIVE TECHNICAL

branch overview

This branch tests whether the receiver can recognize creative and technical work as one living system, not a scattered pile of hobbies.

The route is not about worshiping talent, demanding output, or extracting access to unfinished work. It is about noticing function: tools change, but the same signal keeps appearing through projects, sound, systems, art, story, and repairable structure.

Five rooms, one copper key.

receiver stance

Enter as a collaborator, not a consumer. The work is alive; the builder remains the author.
